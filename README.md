##В данном репозитарии хранятся 12 проектов созданных в процессе обучения на специалиста по Data Science от Я.Практикума
Темы проектов:
1. Исследовательский анализ данных. Исследование объявлений о продаже квартир
2. Статистический анализ данных. Проверка гипотез для сервиса аренды самокатов
3. Сборный проект. Закономерности определяющие успешность игры
4. Линейные модели в машинном обучении. Подбор бурёнок для молочного хозяйства
5. Обучение с учителем: качество модели. Персонализация предложений для клиентов интернет-магазина
6. Сборный проект-2. HR-аналитика
7. Машинное обучение в бизнесе. Выбор локации для скважины
8. Численные методы. Определение рыночной стоимости автомобиля
9. Временные ряды. Прогноз заказов такси
10. Машинное обучение для текстов. Модель для классификации комментариев
11. Компьютерное зрение. Определение возраста покупателей
12. Выпускной проект. Прогноз оттока клиентов
